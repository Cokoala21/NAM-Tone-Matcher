▶ Setup instructions
1. Register your redirect URI on TONE3000
Go to tone3000.com/settings and add your redirect URI. If running locally:
http://localhost:8080/nam-tone-matcher.html

2. Serve this file locally
In the same folder as this file, run:
python3 -m http.server 8080
Then open http://localhost:8080/nam-tone-matcher.html

3. Click "Connect TONE3000" — you'll be redirected to log in, then brought back automatically.

Loading in REAPER
01 — Download assets
Download the selected .nam and .wav IR from the TONE3000 links. Save to a known folder.
02 — Install NAM plugin
Install Neural Amp Modeler VST3 by Steven Atkinson from github.com/sdatkinson/NeuralAmpModelerPlugin.
03 — Import template
In REAPER: File → Track Templates → Load Track Template, select the .RTrackTemplate.
04 — Load .nam model
Open the NAM plugin on the track. Click the folder icon to load your downloaded .nam file.
05 — Load IR
Open the ReaVerb instance and load your downloaded cabinet .wav IR file.
06 — Dial in
Adjust FX parameters to taste. Template defaults match the original recording context.
